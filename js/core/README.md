# Ядро приложения (Core)

1. Бизнес-функция: Точка входа, хранилище, роутер. Не зависит от фич.
2. Структура:
   - app.js — Инициализация приложения.
   - store.js — Единое хранилище состояния.
   - router.js — Маршрутизация страниц.
   - events.js — Шина событий.
3. Связи: Ядро → Все фичи (через store и события).
