# Логика комнат

1. Бизнес-функция: Сборка, склейка, валидация геометрии комнат.
2. Структура:
   - detect-room.js — Поиск замкнутого контура среди линий.
   - dfs-utils.js — Алгоритмы поиска циклов и соединений.
   - room-builder.js — Создание объекта Комнаты.
   - render-room.js — Отрисовка комнаты (заливка, стены, подписи).
   - info-box.js — Инженерное окошко (площади, периметр).
   - room-label.js — Управление названием комнаты.
   - stitch.js — Кнопка «Собрать» (склейка нескольких контуров).
   - wall-geometry.js — Расчёт координат при изменении стены.
3. Связи: draw-line.js → detect-room.js → room-builder.js → State + Render.
