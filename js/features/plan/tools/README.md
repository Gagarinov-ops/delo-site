# Инструменты модуля «План»

1. Бизнес-функция: Панель управления и взаимодействие с пользователем.
2. Структура:
   - toolbar.js — Панель инструментов (Курсор, Линия, Элементы, Отменить).
   - element-library.js — Каталог готовых элементов (полоски, уголок).
   - drag-shape.js — Перетаскивание готовых линий.
   - input-dispatcher.js — Обработчик кликов/тапов.
3. Связи: Панель → Диспетчер → Инструменты → Actions.
