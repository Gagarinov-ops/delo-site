# Модуль «План помещения»

1. Бизнес-функция: Создание плана помещения, расчёт площадей и передача данных в смету.
2. Структура:
   - core/ — Фундамент (состояние, действия, сетка, рисование).
   - tools/ — Инструменты (тулбар, каталог элементов, drag-and-drop, ввод).
   - room/ — Логика комнат (детектор, строитель, отрисовка, расчёт).
   - geometry/ — Геометрическое ядро (хранение точек, солвер, коннектор).
   - render.js — Отрисовка всех фигур на canvas.
   - wall-input.js — Поле ввода реальной длины стены.
3. Связи: Инструменты → Actions → State → Render → InfoBox → Смета (через store.plan).
