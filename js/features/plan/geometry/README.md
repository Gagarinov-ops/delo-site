# Геометрическое ядро

1. Бизнес-функция: Хранение точек (joints), пересчёт координат, стыковка.
2. Структура:
   - state.js — Хранилище точек (joints) комнат.
   - solver.js — «Мозг»: пересчёт координат с учётом допусков.
   - connector.js — Мост между кликами пользователя и солвером.
3. Связи: wall-input.js → connector.js → solver.js → state.js → render-room.js.
