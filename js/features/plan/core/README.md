# Фундамент модуля «План»

1. Бизнес-функция: Базовые кирпичи, не зависящие от надстроек.
2. Структура:
   - state.js — Склад данных (линии, комнаты).
   - actions.js — Единственная рука, меняющая склад.
   - grid.js — Сетка и координатная плоскость.
   - draw-line.js — Инструмент «Карандаш».
3. Связи: Инструменты → Actions → State → Render.
