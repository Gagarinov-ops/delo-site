# Модуль «Смета»

## Структура
- state.js — хранилище позиций сметы
- actions.js — диспетчер операций (добавить, удалить, обновить)
- table.js — таблица позиций с кнопками удаления
- form-validation.js — валидация полей и добавление позиции
- form.js — форма добавления позиции
- summary.js — блок итоговой суммы
- connector.js — подключение к данным Плана
