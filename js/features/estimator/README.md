# Модуль «Смета»

1. Бизнес-функция: Приём данных из плана, отображение сметы, ручной ввод позиций.
2. Структура:
   - state.js — Хранилище позиций сметы.
   - actions.js — Диспетчер операций (добавить, удалить, обновить).
   - table.js — Таблица позиций.
   - form.js — Форма добавления позиций.
   - form-validation.js — Валидация полей формы.
   - summary.js — Блок итоговой суммы.
   - connector.js — Приём данных из store.plan.
3. Связи: store.plan → connector.js → actions.js → state.js → table.js + summary.js.
